MT5 Connector

Connect live data through the EA and Google Apps Script

LIVE
1

Put `GoldDashboardExporter.mq5` in `MQL5/Experts`, then compile it into `.ex5`.

2

Enable WebRequest in MT5, paste the Apps Script Web app URL into `InpWebhookUrl`, then add `InpWriteKey`.

3

The EA sends snapshots into Google Sheets, then the dashboard fetches live data instead of demo data.

Public demo page. Real dashboard data requires read key. No MT5 passwords or trading credentials are stored in this repository.
